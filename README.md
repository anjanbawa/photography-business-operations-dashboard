# Photography Business Operations Dashboard

**Tool:** Microsoft Excel | **Domain:** Business Operations & Project Management  
**Context:** The Timeless Pictures — Jamshedpur, India (2025)

---

## Business Problem

Running a photography business with 14 client projects, a 3-person team, and multiple vendor relationships generates a lot of operational data — but without a centralized tracking system, it's impossible to answer critical questions:

- Which months are peak demand periods requiring more team members?
- Which project types generate the highest revenue and profit?
- Are projects being delivered on time?
- How is each team member performing across projects?

This dashboard was built to answer all of these questions from a single Excel file using real client invoice data.

---

## What the Dashboard Does

### Sheet 1 — Project Data
Real data from 14 actual client invoices including client name, project type, assigned team member, bill amount, actual cost, profit, and project status.

### Sheet 2 — KPI Dashboard
8 live KPI cards giving an instant snapshot of business health:
- Total projects completed
- Total revenue billed and total profit
- Average profit margin
- Highest value single project
- Average project value

### Sheet 3 — Monthly Analysis
Month-by-month breakdown of billing, cost, and profit with a dynamic bar chart. Jun–Jul show zero bookings — a real operational insight for advance planning.

### Sheet 4 — Team Performance
Team-level performance tracker using SUMIF/COUNTIF formulas to automatically calculate each member's total billing, cost, profit, and margin — pulling live from the raw data sheet.

### Sheet 5 — Project Type Analysis
Revenue breakdown by project type (Cinematic Wedding, Traditional Wedding, Engagement Shoot, etc.) with bar chart to identify highest-value service categories.

## Technical Skills Demonstrated

- **SUMIF / COUNTIF / AVERAGEIF** — dynamic aggregation across sheets
- **Cross-sheet formula references** — all KPIs pull live from raw data
- **IF formulas** — handling empty months without errors
- **Bar charts** — monthly billing vs cost, and revenue by project type
- **Clean data structure** — invoice-level granularity, ready for pivot analysis
- **Excel best practices** — no hardcoded calculated values, zero formula errors

## How to Use

1. Open `The_Timeless_Pictures_Operations_Dashboard_2025.xlsx`
2. Start with **KPI Dashboard** for a business overview
3. Use **Monthly Analysis** to identify demand trends and plan team availability
4. Check **Team Performance** to evaluate workload distribution
5. Use **Project Type Analysis** for pricing and service strategy decisions
6. All data can be updated in **Project Data** — all other sheets recalculate automatically
